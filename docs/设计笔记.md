## 数据结构

### Table

- 表由页组成
    - 一页4096B=4KB
    - 一个表设计为256页，=1MB
    - 页用于存放tuple
    - 数据文件格式为`.dat`
- 需要另外一个文件用于存放schema信息
    - 格式为`.inf`

## 指令执行流程

### create table流程

- 创建对应`stmt`
- `Parser::parse_sql`  解析 ->转进`Parser::parse_create`, 返回带有schema信息的`c_stmt`
    - schema中包含列信息，与表名
- 将`c_stmt`和`db_name`传进`c_executor`中,创建一个表
